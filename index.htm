Problem Statement: "Next-Gen Address Verification & Fraud Detection in Heterogeneous Datasets"

The Challenge Landscape

The Unique Identification Authority of India (UIDAI) manages over 1.3 billion identities. The 2026 Hackathon focuses on a critical bottleneck: Address Verification. With India's diverse linguistic landscape and non-standardized address formats, automated verification often fails, leading to manual reviews and potential fraud. The goal is to build automated, privacy-preserving systems to clean, deduplicate, and verify address data at scale.

Impact of Manual Intervention

When automation fails, manual verification is triggered. This drastically increases the Turnaround Time (TAT) and operational cost.

Solution 1: "IndicBERT" Address Normalization

Concept: A transformer-based NLP model specifically trained on Indian address datasets. Unlike standard regex or string matching, this model uses semantic embeddings to understand that "Opposite Civil Hospital" and "Near Gov. Hospital" refer to the same locality.

Solution 3: Privacy-First Verification

Zero Knowledge Proofs (ZKP)

Instead of sharing the actual address data with third-party service providers (TSPs), the system generates a cryptographic proof.

"Does User X live in Zipcode 110001?"
Answer: YES (without revealing the house number or street).

Tech Stack:

  • ZoKrates: Toolbox for zkSNARKs on Ethereum/Blockchain.
  • Circom: Circuit compiler for defining constraints.
  • Off-Chain Storage: IPFS for encrypted document hashes.

Implementation Roadmap

A phased approach to deploying these solutions within the hackathon timeframe and beyond.

Phase 1: Data Pre-processing (Weeks 1-4)

Cleaning historical data, training the IndicBERT model on address variations, and establishing baseline error metrics.

Phase 2: Pilot Deployment (Weeks 5-8)

Deploying the API wrapper for the semantic matcher. Running shadow verification (running alongside the old system to compare results).

Phase 3: Integration & Scale (Months 3-6)

Full integration with the UIDAI enrollment client. Enabling Geospatial flagging on live incoming streams.
